# raylib_tcp_messaging_app
Simple tcp messaging app using raylib

## Build and run application

### Build
<pre>git clone https://github.com/GBU31/raylib_tcp_messaging_app.git</pre>
<pre>mkdir build</pre>
<pre>cd build</pre>
<pre>cmake ..</pre>
<pre>make</pre>

### Run
<pre>./server</pre>
<pre>./client</pre>
